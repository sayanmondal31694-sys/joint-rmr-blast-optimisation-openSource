# Joint-RMR Blast Design Optimisation — Open Source Replication

Replication of Chandrahas & Sahu (2025) blast optimisation framework using 
entirely free tools: CloudCompare (FACETS plugin) and Python (mplstereonet).

## Study site
Villamanin limestone quarry, León, Spain (Escavy et al., 2022 dataset)

## What this repo contains
- `Villamanin_Blast_Analysis_Report_SayanMondal.docx` — full methodology report
- `blast_analysis.ipynb` — Python notebook: joint clustering, RMR calculation, 
   blast performance prediction, and visualisation

## Key results
- 4,683 joint facets extracted from 500,000-point cloud
- 6 joint families identified via stereonet analysis
- RMR: 52 (unfavourable orientation) → 72 (favourable orientation)
- Projected backbreak reduction: 64.7% by optimising firing direction

## Tools used
- CloudCompare v2.13.2 with FACETS plugin (free)
- Python: pandas, numpy, mplstereonet, matplotlib, scipy
